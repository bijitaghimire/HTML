HTML is defined as a markup language and a way to indicate information about the content.HTML 5 is widely implemented in modern browsers. The W3C is relatively new and not always taught.HTML SYNTAX consists of HTML documents and HTML elements. The types of nesting includes correct and incorrect nesting and its structure ranges from simple to complex and consist of three important elements ie html,head and body that contains description and body of contents.it contains 6 level of headings.HTML elements consists of links,tags URL relative referencing, inline text element, images,lists character entities and so on.HTML elements consists of ordered , unordered and description list.Tools insight contains integrated development environment, Code editors ,cloud based environment and cloud playground.CSS is a computer language for laying out and structuring web pages,it describes and assign font properties, colour, sizes ,borders, background etc.It improves accessibility, maintainability , flexibility and controls over formatting although the consistency was lacking previously.A CSS document consists of one or more styles,it contains a selector and a series of property value pairs.The selector identifies the elements of the document and each individual CSS must contain a property,and the unit of value must be dependent upon the property. The two major units are relative and absolute unit. Talking about the location of style,the inline style lies within the HTML element.Embedded style sheets (also called internal styles) are style rules placed within the element.External style sheets are style rules placed within a external text file with the css extension .The types of selectors include Elements selectors containing all HTML elements. A class selector allows you to simultaneously target different HTML elements regardless of their position .ID selector targets specific elements whereas an attribute selector provides a way to select HTML elements either by the presence of an element attribute or its value.The page construction can be done by CSS or HTML and the elements are header,nev,article,aside and footer,and fluid design for the layout.
