lecture1-internet technology and web development.

Internet Protocol (IP) in general is knownas a set of rules that dictate how data should be delivered over the public network (Internet).These protocols have been implemented in every operating System for fast web development .Web developer needs general awareness of what the suite of Internet protocol does. General idea of of the hardware component of the Internet is that which we experience in our homes.The wireless router is perhaps the most visible Manifestation of the Internet in one’s home, in that it Is a device we typically need to purchase and install. It’s a hardware device that forwards data packets from one network to another.The scale varies from small to large depending upon the area of network that includes out of the house, routing tables from local provider to ocean edge etc.Eventually, international Internet communication will need to travel underwater.The Domain Name System (DNS) is the Internet’s system for mapping alphabetic names to numeric Internet Protocol (IP) addresses. .The type of top level domain are generic top level domain (gTLD) country code top level domain (ccTLD).The internet and www are different but related things.The advantages includes Storage, accessibility, security etc and disadvantages such as internet problem, restriction of different operative resources and mishandling of personal datas.URL (Uniform Resource Locators) are the Web site addresses you enter into the address bar of your browser,each valid URL points to a unique resource. Such resources can be an HTML page, a CSS document, an image, etc.Hypertext Transfer Protocol ( HTTP) is a communications protocol. It is used to send and receive webpages and files on the internet,eg:200: OK.Its features consists of search engine integration, URL autocompletion, Form autocompletion etc.A web server is s computer that responds to the HTTP request ,the strength varies from real world servers to home desktop. It must consist of application stack operating system, web server software, a database and a scripting language for dynamic requests. It mainly relies on LAMP software stack. Web development is the work involved in developing a Web site for the Internet or an intranet .It is like a complicated ecosystem .The web server model consist of the request-response loop,the peer to peer alternative. Roles and skills includes of software engineering,Business Analyst, Nontechnical Roles,SEO specialist etc.
